    • Escriba instrucciones que asignen enteros aleatorios a la variable n en los siguientes rangos:

    1. 1 ≤ n ≤ 2

      2. n = Math.floor(Math.random() * (2) + 1);

    2. 1 ≤ n ≤ 100

      3. n = Math.floor(Math.random() * (100 - 1) + 1);

    3. 0 ≤ n ≤ 9

      4. n = Math.floor(Math.random() * (9 - 0) + 0);

    4. 1000 ≤ n ≤ 1112

      5. n = Math.floor(Math.random() * (1112 - 1000) + 1000);

    5. -1 ≤ n ≤ 1

      6. n = Math.floor(Math.random() * (2 + 1) - 1);

    6. -3 ≤ n ≤ 11

      7. n = Math.floor(Math.random() * (-3) + 11);
