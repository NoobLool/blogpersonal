• Inicio

    • Los conductores se preocupan acerca del kilometraje de sus automóviles. Un conductor ha llevado el control de varios reabastecimientos de gasolina, registrando los kilómetros conducidos y los litros usados en cada viaje. Desarrolle una secuencia de comandos que reciba como entrada los kilómetros conducidos y los litros usados por cada viaje. La secuencia de comandos debe calcular y producir texto de HTML5 que muestre el número de kilómetros por litro obtenidos en cada viaje, y debe imprimir el total de kilómetros por litro obtenidos en todos los reabastecimientos hasta este punto. Use diálogos prompt para obtener los datos del usuario.

    Pseudocódigo:
    Variables kilometros = 1, gasolina = 0, kilPorLit, total = 0, viaje = 1;
    Abrir etiqueta h3
    Mientras la cantidad de viaje sea menor o igual a 50:
      Recibir kilometros conducido en el viaje
      Convertir kilometros en entero
      Si kilometros es igual a 0
        Romper el ciclo Mientras
      Recibir gasolina usados en el viaje
      Convertir gasolina en entero
      Calcular kilometros recorridos por cada litro de gasolina en el viaje
      Acumular total de kilometros por litro en todos los viajes
      Imprimir resultado
      Incrementar viaje
    Si viaje es mayor a 0
      Imprimir total de kilometros por litro en todos los viajes
    Cerrar etiqueta h3

    Código JavaScript:
    let kilometros = 1, gasolina = 0, kilPorLit, total = 0, viaje = 0;
    document.writeln("<h3>");
    while(viaje <= 50){
      kilometros = window.prompt("Kilómetros conducidos (viaje " + (viaje + 1) +")\nNota: Introduzca 0 para salir.", 0);
      kilometros = parseInt(kilometros);

      if(kilometros == 0) break;

      gasolina = window.prompt("Gasolina usados (viaje " + (viaje + 1) +")", 0);
      gasolina = parseInt(gasolina);

      kilPorLit = kilometros / gasolina;

      total += kilPorLit;

      document.writeln("Kilómetros por litro (viaje " + viaje + "): " + kilPorLit + "<br>");

      ++viaje;
    }
    viaje != 0 ? document.writeln("Total de kilómetros por litros: " + total) : null;
    document.writeln("</h3>");
