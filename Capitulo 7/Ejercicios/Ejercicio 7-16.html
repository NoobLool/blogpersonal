    • Escriba una secuencia de comandos que utilice ciclos para imprimir la siguiente tabla de valores. Muestre los resultados en una tabla de HTML6. Use CSS para centrar los datos en cada columna.

    Pseudocódigo:
    Variable contador = 1;
    Abrir etiqueta h3 y center
    Abrir etiqueta table e indicar el border y text-align
    Abrir etiqueta thead
    Abrir etiqueta tr
    Abrir etiqueta th con todos los nombres de las columnas
    Cerrar etiqueta tr
    Cerrar etiqueta thead
    Abrir etiqueta tbody
    Mientras contador sea menor a 7
      Abrir etiqueta tr
      Abrir y Cerrar etiqueta td con el valor de contador
      Abrir y Cerrar etiqueta td con el producto de contador * 10
      Abrir y Cerrar etiqueta td con el producto de contador * 100
      Abrir y Cerrar etiqueta td con el producto de contador * 1000
      Cerrar etiqueta tr
      Incrementar contador
    Cerrar etiqueta tbody
    Cerrar etiqueta table
    Cerrar etiqueta h3 y center

    Código JavaScript:
    let contador = 1;
    document.writeln("<h3><center>");
    document.write("<table border='1' style='text-align: center;'>");
    document.write("<thead>");
    document.write("<tr>");
    document.write("<th>N</th><th>10*N</th><th>100*N</th><th>1000*N</th>");
    document.write("</tr>");
    document.write("</thead>");
    document.write("<tbody>");
    while ( contador < 7 ){
      document.write("<tr style='font-weight: normal;'>");
      document.write("<td>" + contador + "</td>");
      document.write("<td>" + (contador * 10) + "</td>");
      document.write("<td>" + (contador * 100) + "</td>");
      document.write("<td>" + (contador * 1000) + "</td>");
      document.write("</tr>");
      ++contador;
    }
    document.write("</tbody>");
    document.write("</table>");
    document.writeln("</h3></center>");
