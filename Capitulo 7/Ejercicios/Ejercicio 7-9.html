    • Identifique y corrija los errores en cada uno de los siguiente segmentos de código. [Nota: puede haber más de un error en cada fragmento; a menos que haya declaraciones presentes, suponga que todas las variables están declaradas e inicializadas correctamente]

    1. if ( edad >= 65 );
        document.writeln( "Edad es mayor o igual que 65" );
      else
        document.writeln( "Edad es menor que 65" );

      2. El error se encuentra después de la condición del if, ya que tiene un ' ; ', lo que hace que se ejecute la siguiente línea sin importar si la condición se cumple o no.
      Correción:
      if ( edad >= 65 )
        document.writeln( "Edad es mayor o igual que 65" );
      else
        document.writeln( "Edad es menor que 65" );

    2. var x = 1, total;
      while ( x <= 10 )
      {
        total += x;
        ++x;
      }

      3. El error se encuentra en la declaración de la variable total, ya que debe de indicarse su valor inicial que tendrá.
      Correción:
      var x = 1, total = 0;
      while ( x <= 10 )
      {
        total += x;
        ++x;
      }

    3. var x = 1;
      var total = 0;
      while ( x <= 100 )
        total += x;
        ++x;

      4. El error es que no se están utilizando '{}' para el cuerpo del while.
      Correción:
      var x = 1;
      var total = 0;
      while ( x <= 100 )
      {
        total += x;
        ++x;
      }

    4. var y = 5;
      while ( y > 0 )
      {
        document.writeln( y );
        ++y;

      5. El primer error se encuentra en la condicón del while ya que siempre será verdadera, por lo que serí un ciclo infinito.
      El segundo error es que no se está cerrando el bloque while con '}'.
      Posible correción:
      var y = 5;
      while ( y > 0 )
      {
        document.writeln( y );
        --y;
      }
