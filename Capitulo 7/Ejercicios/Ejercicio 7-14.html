• Inicio

    • Desarrolle una secuencia de comandos que determine el suelo bruto para cada uno de tres empleados. La empresa paga la "cuota normal" en las primeras 40 horas de trabajo de cada empleado, y paga "cuota y media" en todas las horas trabajadas que excedan de 40. Usted recibe una lista de los empleados de la empresa, el número de horas qu trabajó cda uno la semana pasada y la tarifa por horas de cada empleado. Su programa debe recibir como entrada esta información para cada empleado, determinar el sueldo bruto de cada uno y producir texto de HTML5 que muestre ese sueldo bruto. Utilice diálogos prompt para introducir los datos.

    Pseudocódigo:
    Variables empleado = 0, horas = 0, tarifa = 0, sueldoBruto = 0;
    Abrir etiqueta h3
    Mientras empleado sea menor a 3
      Recibir horas trabajadas del empleado
      Convertir horas en entero
      Recibir tarifa por hora del empleado
      Convertir tarifa en entero
      Si horas es menor o igual a 40
        Calcular sueldoBruto es igual a horas por tarifa
      Si-no     Calcular sueldobruto es igual a tarifa por 40 horas
        Acumular sueldoBruto con el producto de (horas - 40) y (tarifa * 1.5)
      Imprimir sueldo bruto del empleado
      Incrementar empleado
    Cerrar etiqueta h3

    Código JavaScript:
    let empleado = 0, horas = 0, tarifa = 0, sueldoBruto = 0;
    document.writeln("<h3>");
    while(empleado < 3){
      horas = window.prompt("Horas trabajadas del empleado " + (empleado + 1), 40);
      horas = parseInt(horas);

      tarifa = window.prompt("Tarifa por hora del empeado " + (empleado + 1), 1);
      tarifa = parseFloat(tarifa);

      if(horas <= 40) sueldoBruto = horas * tarifa;
      else{
        sueldoBruto = tarifa * 40;
        sueldoBruto += (horas - 40) * (tarifa * 1.5);
      }

      document.writeln("Sueldo bruto del empleado " + (empleado + 1) + " es: " + sueldoBruto + "<br>");

      ++empleado;
    }
    document.writeln("</h3>");
