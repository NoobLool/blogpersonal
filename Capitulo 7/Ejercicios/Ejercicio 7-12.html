• Inicio

    • Desarrolle una secuencia de comandos que determine si alguno de los clientes de una tienda de departamentos se ha excedido del límite de crédito de una cuenta. Para cada cliente se tienen los siguientes datos:
    a) El número de cuenta.
    b) El saldo al inicio del mes.
    c) El total de todos los artículos cargados por el cliente en el mes.
    d) El total de todos los créditos aplicados a la cuenta del cliente en el mes.
    e) El límite de crédito permitido.
    La secuencia de comandos debe recibir como entrada cada uno de estos datos desde un diálogo prompt como entero, debe calcular el nuevo saldo (= saldo inicial + cargos - créditos), mostrar el nuevo balance y determinar si éste excede el límite de crédito del cliente. Para los clientes cuyo límite de crédito sea excedido, la secuencia de comandos debe producir texto de HTML5 que muestre el mensaje "Se excedió el límite de su crédito".

    Pseudocódigo:
    Variables noCuenta = 1, saldoIn = 0, cargos = 0, creditos = 0, limite = 0, nuevoSaldo, clientes = 0;
    Abrir etiqueta h3
    Mientras clientes sea menor igual a 50:
      Recibir número de cuenta
      Convertir noCuenta en entero
      Si noCuenta es igual a 0
        Romper el ciclo Mientras
      Recibir total de cargos del mes
      Convertir cargos en entero
      Recibir total de créditos
      Convertir creditos en entero
      Recibir límite de créditos
      Convertir creditos en entero
      Calcular nuevo saldo
      Imprimir noCuenta, nuevoSaldo y limite del cliente
      Si nuevoSaldo es mayor a limite
        Imprimir mensaje de que excedió del límite
      Incrementar clientes
    Cerrar etiqueta h3

    Código JavaScript:
    let noCuenta = 1, saldoIn = 0, cargos = 0, creditos = 0, limite = 0, nuevoSaldo, clientes = 0; document.writeln("<h3>");
    while(clientes <= 50){
      document.writeln("<br>");
      noCuenta = window.prompt("Número de cuenta\nNota: Introduzca 0 para salir.", 0);
      noCuenta = parseInt(noCuenta);

      if(noCuenta == 0) break;

      saldoIn = window.prompt("Saldo inicial del mes", 1);
      saldoIn = parseInt(saldoIn);

      cargos = window.prompt("Total de cargos del mes", 1);
      cargos = parseInt(cargos);

      creditos = window.prompt("Total de créditos del mes", 1);
      creditos = parseInt(creditos);

      limite = window.prompt("Límite de crédito", 1);
      limite = parseInt(limite);

      nuevoSaldo = saldoIn + cargos - creditos;

      document.writeln("Cuenta: " + noCuenta + "<br>Nuevo saldo: " + nuevoSaldo + "<br>Límite: " + limite + "<br>");
      if( nuevoSaldo > limite )
        document.writeln("Se excedió el límite de su crédito.");
      ++clientes;
    }
    document.writeln("</h3>");
