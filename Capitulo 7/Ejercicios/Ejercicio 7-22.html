• Inicio

    • Una compañía desea transmitir datos por teléfono, pero le preocupa que sus teléfonos puedan estar intervenidos. Todos sus datos se transmiten como enteros de cuatro dígitos. La compañía le ha pedido a usted que escriba una secuencia de comandos que cifre sus datos, de modo que puedan transmitirse en forma más segura. Su secuencia de comandos debe leer un entero de cuatro dígitos introducido por el usuario en un diálogo prompt y cifrarlo de la siguiente manera: reemplace cada dígito con (la suma de ese dígito más 7) módulo 10. Después intercambie el primer dígito con el tercero, e intercambie el segundo dígito con el cuarto. Luego produzca texto de HTML5 que muestre el entero cifrado.

    Pseudocódigo:
    Variables numero, digito4, digito3, digito2, digito1, temp, cifrado
    Recibir numero entero de cuatro dígitos
    Convertir numero en entero
    Mientras al dividir numero / 1000 el resultado es menor a 1 o es mayor igual a 10 ←No es de cuatro dígitos
      Mostrar alerta con el mensaje error
      Recibir nuevo valor de numero
      Convertir numero en entero
    Obtener primer dígito de numero con la operación (numero/1000), y convertirlo a entero
    Obtener segundo dígito de numero con la operación (numero/100 % 10), y convertirlo a entero
    Obtener tercer dígito de numero con la operación (numero/10 % 10), y convertirlo a entero
    Obtener cuarto dígito de numero con la operación numero%10
    Calcular la primera parte del cifrado ( digito1 = [digito1 + 7] % 10 )
    Calcular la primera parte del cifrado ( digito2 = [digito2 + 7] % 10 )
    Calcular la primera parte del cifrado ( digito3 = [digito3 + 7] % 10 )
    Calcular la primera parte del cifrado ( digito4 = [digito4 + 7] % 10 )
    Intercambiar digito1 con digito3 con ayuda de un temporal
    Intercambiar digito2 con digito4 con ayuda de un temporal
    Unir los cuatro dígitos en la variable cifrado
    Imprimir número cifrado

    Código JavaScript:
    let numero, digito4, digito3, digito2, digito1, temp, cifrado;
    numero = window.prompt("Escriba un número entero de 4 dígitos", "1111");
    numero = parseInt(numero);
    while( ((numero/1000) < 1) || ((numero/1000) >= 10) ){
      window.alert("El numero entero no es de 4 dígitos.");
      numero = window.prompt("Escriba un número entero de 4 dígitos", "1111");
      numero = parseInt(numero);
    }

    digito1 = parseInt(numero/1000);
    digito2 = parseInt(numero/100 %10);
    digito3 = parseInt(numero/10 %10);
    digito4 = numero%10;

    digito1 += 7 % 10;
    digito2 += 7 % 10;
    digito3 += 7 % 10;
    digito4 += 7 % 10;

    temp = digito3;
    digito3 = digito1;
    digito1 = temp;

    temp = digito4;
    digito4 = digito2;
    digito2 = temp;

    cifrado = digito1 + "" + digito2 + "" + digito3 + digito4;

    document.write("<h3>Número entero cifrado: " + cifrado + "</h3>");
